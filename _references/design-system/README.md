# Design System Reference

Place your Flutter design system files here:
- app_colors.dart
- app_typography.dart
- app_dimensions.dart
- app_shadows.dart
- app_theme.dart
- Any Figma-exported JSON token files
- Color palette screenshots or swatches

The agent will read these to extract exact color values, spacing scales,
typography scales, and shadow definitions for use in tailwind.config.ts
and globals.css.
