## Overview

Tazrout is a smart irrigation system built on a multi-layer architecture...

## Layers

1. **Field Layer** — ESP32 microcontrollers + soil/climate sensors + LoRa transceivers
2. **Gateway Layer** — Raspberry Pi running MQTT broker + local backend
3. **Application Layer** — Spring Boot backend + PostgreSQL database
4. **Presentation Layer** — Flutter mobile/desktop dashboard + this website
